# 🎓 跟着弹模式 - 使用指南

## 📖 什么是跟着弹模式？

跟着弹模式是专为零基础用户设计的互动式钢琴学习功能。它通过**实时高亮提示**和**即时反馈**，让您一步一步学会弹奏钢琴曲目。

## ✨ 核心特点

### 🎯 零基础友好
- **无需任何音乐基础**，只需跟着提示弹奏
- **高亮提示**：下一个要弹的琴键会自动高亮
- **即时反馈**：弹对显示"✓"，弹错提示"再试一次"
- **必须弹对才能继续**：确保每一步都学会

### 📊 进度可视化
- 实时显示当前进度（如：5 / 30）
- 百分比进度条直观展示
- 完成后可选择其他歌曲继续练习

### 🎹 操作方式
- **鼠标点击**：直接点击屏幕上的琴键
- **键盘按键**：使用电脑键盘对应按键
  ```
  白键：a s d f g h j k l
  黑键：w e   t y u   o p
  ```

## 🚀 如何使用

### 步骤 1：打开页面
访问 `follow-along.html` 文件，或者在主页点击"跟着弹"链接

### 步骤 2：选择歌曲
从下拉菜单中选择一首想要学习的歌曲：
- 简单爱 - 周杰伦
- 告白气球 - 周杰伦
- 七里香 - 周杰伦
- 成都 - 赵雷
- 纸短情长 - 烟把儿
- 千与千寻 - 久石让
- One Summer's Day
- Canon in D - 卡农

### 步骤 3：开始练习
1. 点击"开始练习"按钮
2. 看到高亮的琴键
3. 用鼠标点击或按对应的键盘按键
4. 弹对了会自动进入下一个音符
5. 弹错了可以继续尝试

### 步骤 4：控制功能
- **暂停**：随时可以暂停练习
- **继续**：从暂停处继续
- **重新开始**：从头开始当前歌曲

## 💡 学习建议

### 🌟 对于完全零基础：
1. **从简单的歌曲开始**：推荐"千与千寻"或"Canon"
2. **不要着急**：慢慢来，确保每个音符都弹对
3. **多重复练习**：同一首歌曲可以练习多次
4. **先记住位置**：熟悉常用音符的键盘位置

### 🎯 进阶技巧：
1. **逐步减少看键盘**：尝试通过手感记住位置
2. **提高速度**：熟练后可以更快地弹奏
3. **尝试更多歌曲**：扩展曲目库
4. **组合练习**：配合"手动练习"模式一起使用

## ❓ 常见问题

**Q: 为什么只有30个音符？**
- A: 为了让零基础用户不会感到挫败，精选了每首歌最核心的30个音符作为练习内容。熟练后可以使用完整的"手动练习"模式。

**Q: 弹错了怎么办？**
- A: 没关系！可以一直尝试，直到弹对为止。不会跳过任何一个音符。

**Q: 可以调速度吗？**
- A: 跟着弹模式由您控制节奏，想多慢就多慢。没有时间限制。

**Q: 手机上能用吗？**
- A: 完全可以！支持触摸操作，建议横屏使用体验更好。

**Q: 音频加载慢怎么办？**
- A: 首次打开需要加载钢琴音频采样，建议在WiFi环境下使用。加载完成后就很流畅了。

## 🎉 学习路径建议

### 第1阶段：入门（第1-3天）
1. 打开跟着弹模式
2. 选择"千与千寻"（最简单）
3. 完成前10个音符
4. 重复练习直到不看键盘也能弹

### 第2阶段：熟悉（第4-7天）
1. 完成"千与千寻"全部30个音符
2. 尝试"Canon"
3. 开始记忆常用音符位置
4. 提高弹奏速度

### 第3阶段：进阶（第2周）
1. 尝试所有可选歌曲
2. 配合"手动练习"模式
3. 学习完整曲目
4. 尝试自动播放模式观察

### 第4阶段：自由（第3周+）
1. 不依赖提示弹奏
2. 尝试MIDI播放
3. 探索更多曲目
4. 享受音乐乐趣！
