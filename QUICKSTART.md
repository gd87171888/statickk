# 🚀 快速开始 | Quick Start

## 5分钟上手指南

### 1️⃣ 启动项目 (3种方式)

#### 方法一：使用启动脚本（推荐）

**Linux/Mac:**
```bash
./start.sh
```

**Windows:**
```bash
start.bat
```

#### 方法二：使用Python
```bash
# Python 3
python3 -m http.server 8080

# Python 2
python -m SimpleHTTPServer 8080
```

#### 方法三：使用Node.js
```bash
npx serve -p 8080 .
```

#### 方法四：直接打开
```bash
# 双击 index.html 文件
# 或拖动到浏览器
```

### 2️⃣ 访问应用

打开浏览器访问：
- 🌐 本地: http://localhost:8080
- 📱 手机: http://[你的电脑IP]:8080

---

## 📱 移动端访问

### 同一WiFi下访问

1. **获取电脑IP地址**
   ```bash
   # Mac/Linux
   ifconfig | grep "inet "
   
   # Windows
   ipconfig
   ```

2. **手机访问**
   - 打开手机浏览器
   - 输入: `http://192.168.x.x:8080`
   - 替换为你的实际IP地址

3. **添加到主屏幕**（推荐）
   - iOS: Safari → 分享 → 添加到主屏幕
   - Android: Chrome → 菜单 → 添加到主屏幕

## 🎹 基本操作

### 键盘弹奏
```
白键: A S D F G H J K L
黑键: W E   T Y U   O P
低音: Z X C V B N M
```

### 鼠标/触摸
- 直接点击琴键即可发声
- 支持多点触控（和弦）

### 自动播放
1. 右侧列表选择歌曲
2. 点击播放
3. 欣赏自动演奏
