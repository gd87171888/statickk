# 🎹 在线钢琴 - 使用说明

## 快速开始

### 1️⃣ 打开应用

**电脑端：**
- 双击打开 `index.html` 文件
- 或者部署到Web服务器访问

**手机端：**
- 使用手机浏览器打开网页
- 建议"添加到主屏幕"获得更好体验
- 支持横屏和竖屏模式

### 🔗 三大模式导航

现在所有页面都有统一的导航栏，可以轻松切换：

- 🏠 **主页** - 自由弹奏、MIDI播放、曲谱练习
- 📚 **跟弹模式** - 零基础学习，音符高亮引导
- 🎮 **游戏模式** - 节奏游戏，边玩边学

**桌面端：** 点击顶部导航栏即可切换  
**移动端：** 点击右上角 ☰ 汉堡菜单

> 💡 详细使用说明请查看 [功能整合使用指南](INTEGRATION-GUIDE.md)

### 2️⃣ 基础操作

#### 🎵 自由弹奏
- **鼠标**：直接点击屏幕上的琴键
- **键盘**：使用电脑键盘弹奏
  ```
  白键：A S D F G H J K L
  黑键：W E   T Y U   O P
  低音：Z X C V B N M
  ```

#### 🎼 自动播放
1. 在右侧曲谱列表中选择歌曲
2. 点击歌曲名称开始自动播放
3. 可以暂停、停止、调整速度

#### 📝 练习模式
1. 点击"手动练习"按钮
2. 选择想要练习的曲目
3. 按照提示一步步弹奏
4. 适合初学者学习

#### 🎮 游戏模式（最新推荐！）
1. 点击顶部导航栏的"🎮 游戏模式"
2. 选择一首歌曲和难度（简单/普通/困难）
3. 音符从上方落下，在到达判定线时按键
4. 获得判定：Perfect（+100分）、Good（+50分）、Miss（连击清零）
5. 追求高分、连击和成就！边玩边学，超有趣！

#### 🎓 跟着弹模式（零基础推荐！）
1. 点击顶部导航栏的"📚 跟弹模式"
2. 选择一首想学习的歌曲
3. 琴键会高亮提示下一个要弹的音符
4. 弹对了才能继续，弹错可以重试
5. 完美适合零基础用户学习

#### 🎹 MIDI播放
1. 点击"导入MIDI"按钮
2. 选择MIDI文件（或使用内置的MIDI）
3. 系统自动解析并播放

### 3️⃣ 界面功能

#### 顶部菜单栏
- 🏠 **首页**：返回主页
- 🎵 **曲谱**：查看所有可用曲谱
- 🔗 **友情链接**：相关音乐资源

#### 右侧功能区
- 📜 **自动演奏列表**：选择曲目自动播放
- 📝 **手动练习列表**：分步练习指导
- 🛍️ **推荐商品**：音乐相关产品

#### 底部功能
- 💝 **支持作者**：显示捐赠二维码
- 🌐 **社交分享**：分享到微博、QQ、微信等

### 4️⃣ 移动端优化功能

✅ **触摸弹奏**
- 手指轻触琴键即可发声
- 支持多点触控（和弦）
- 触摸反馈优化

✅ **横屏支持**
- 横屏时钢琴键盘更大
- 布局自动优化
- 更好的弹奏体验

✅ **滑动浏览**
- 左右滑动查看所有琴键
- 上下滑动浏览曲谱
- 流畅的触摸体验

✅ **自适应布局**
- 自动适配手机、平板
- 不同尺寸完美显示
- 性能优化

### 5️⃣ 键盘快捷键

| 功能 | 快捷键 |
|------|--------|
| 播放/暂停 | 空格 |
| 停止 | ESC |
| 上一曲 | ← |
| 下一曲 | → |
| 音量+ | ↑ |
| 音量- | ↓ |

### 6️⃣ 常见问题

**Q: 为什么没有声音？**
- 检查设备音量是否打开
- 确保浏览器允许播放音频
- 刷新页面重试

**Q: 手机上琴键太小怎么办？**
- 将手机横屏使用
- 使用双指缩放调整大小
- 选择合适的视图模式

**Q: 如何添加自己的曲谱？**
- 准备JSON或MIDI格式的曲谱
- 放入对应文件夹
- 刷新页面即可加载

**Q: 支持哪些浏览器？**
- Chrome、Firefox、Safari、Edge（推荐）
- iOS Safari 11+
- Android Chrome 60+

**Q: 可以离线使用吗？**
- 下载整个项目到本地
- 打开index.html即可使用
- 所有资源都已包含

### 8️⃣ 技巧提示

💡 **初学者建议**
1. **想边玩边学？试试"游戏模式"！** 像玩音游一样学钢琴
2. **零基础？先用"跟着弹"模式！** 高亮提示+即时反馈
3. 再使用"手动练习"模式熟悉简单曲目
4. 从慢速开始，逐渐提高速度
5. 多听自动播放，培养乐感
6. 先练习右手旋律，再练习左手伴奏

💡 **进阶技巧**
1. 在游戏模式中挑战困难难度，提升反应速度
2. 使用MIDI导入自己喜欢的曲目
3. 调整播放速度来练习困难片段
4. 利用键盘快捷键提高效率
5. 横屏使用手机获得更好视野

💡 **最佳体验**
1. 使用耳机获得更好音质
2. 在安静环境中练习
3. 定期练习保持手感
4. 分享你的进步到社交媒体

### 9️⃣ 移动端专属提示

📱 **添加到主屏幕**（推荐）

**iOS设备：**
1. 点击Safari底部的"分享"按钮
2. 选择"添加到主屏幕"
3. 命名为"在线钢琴"
4. 完成后可像App一样打开

**Android设备：**
1. 点击Chrome菜单（三个点）
2. 选择"添加到主屏幕"
3. 确认添加
4. 在桌面找到图标打开

📱 **省电模式**
- 不使用时关闭页面节省电量
- 降低屏幕亮度
- 使用深色模式（如果浏览器支持）

📱 **流量节省**
- 首次加载会下载音频文件
- 建议在WiFi环境下使用
- 加载完成后可离线使用
