# 🔗 功能整合使用指南 | Integration Usage Guide

## 简介

在线钢琴平台现已完全整合三大核心模式，通过统一的导航栏实现无缝切换。所有功能都在一个平台上，为您提供一站式的钢琴学习体验！

## 🎯 三大核心模式

### 1. 🏠 主页模式（index.html）
**适合对象：** 有一定基础的用户、自由练习者

**核心功能：**
- 🎹 自由弹奏 - 使用键盘即兴演奏
- 🎼 自动演奏 - 播放内置曲谱和MIDI文件
- 📝 手动练习 - 查看曲谱分步练习
- 🎨 随机壁纸 - 美化界面体验
- 💝 社交分享 - 分享到各大平台

**使用方法：**
1. 打开主页，系统自动加载
2. 点击页面上的钢琴键或使用电脑键盘弹奏
3. 选择曲谱进行自动播放或手动练习
4. 使用导航栏随时切换到其他模式

---

### 2. 📚 跟弹模式（follow-along.html）
**适合对象：** 零基础初学者、系统学习者

**核心功能：**
- 🎯 音符高亮 - 清晰指示下一个要弹的音符
- ✅ 即时反馈 - 弹对显示"正确✓"，弹错提示"再试一次"
- 🔒 强制学习 - 必须弹对才能继续，确保学习质量
- 📊 进度显示 - 实时显示完成百分比
- 🎵 多首歌曲 - 8首精选曲目供选择

**使用方法：**
1. 从导航栏点击"📚 跟弹模式"
2. 选择一首想学的歌曲
3. 点击"开始练习"
4. 看到高亮的琴键，弹奏对应的音符
5. 弹对了会自动进入下一个音符
6. 完成所有音符后可选择其他歌曲

**键盘映射：**
- **白键：** A S D F G H J K L ;
- **黑键：** W E T Y U O P

---

### 3. 🎮 游戏模式（game-mode.html）
**适合对象：** 喜欢挑战、想边玩边学的用户

**核心功能：**
- 🎯 下落音符 - 类似Synthesia/Piano Tiles的游戏体验
- 🏆 判定系统 - Perfect（+100分）、Good（+50分）、Miss（连击清零）
- 🎚️ 三种难度 - 简单、普通、困难，适合不同水平
- 📊 实时统计 - 分数、连击、准确率实时显示
- 🏅 成就系统 - 解锁完美演奏、连击大师等成就
- 💾 最高分记录 - 每首歌每个难度都有独立排行榜

**使用方法：**
1. 从导航栏点击"🎮 游戏模式"
2. 选择一首歌曲
3. 选择难度（首次建议选"简单"）
4. 点击"开始游戏"
5. 倒计时后游戏开始
6. 当音符落到底部判定线时按键
7. 游戏结束查看成绩和成就

**难度说明：**
- **简单：** 较慢的速度，宽松的判定窗口，50个音符
- **普通：** 标准速度和判定，100个音符
- **困难：** 快速下落，严格判定，完整曲目

## 🔗 导航栏使用

### 桌面端
在页面顶部可以看到统一的导航栏：
```
🎹 在线钢琴    |    🏠 主页    📚 跟弹模式    🎮 游戏模式
```

点击任意模式即可跳转，当前页面会高亮显示。

### 移动端
在小屏幕设备上，导航菜单会折叠为汉堡菜单（☰）：
1. 点击右上角的"☰"按钮
2. 菜单展开显示三个模式
3. 点击想去的模式
4. 菜单自动收起并跳转

---

## 📱 移动端使用

所有三个模式都完全支持移动端：

### 触摸操作
- **跟弹模式：** 直接点击屏幕上的钢琴键
- **游戏模式：** 触摸虚拟钢琴键盘或点击下落音符
- **主页模式：** 触摸钢琴键盘弹奏

### 横屏优化
在手机上横屏使用可以获得更好的体验，特别是游戏模式。

### 建议
- 在安静的环境中使用，确保能听到音效
- 首次使用可能需要点击屏幕激活音频
- 建议使用耳机获得最佳音质

## 🎓 学习路径推荐

### 零基础用户（推荐）
```
1️⃣ 跟弹模式（2-3天）
   └─ 熟悉键盘位置和基本操作
   
2️⃣ 游戏模式-简单难度（1周）
   └─ 培养节奏感和反应能力
   
3️⃣ 主页-手动练习（持续）
   └─ 系统学习完整曲目
   
4️⃣ 游戏模式-普通/困难（进阶）
   └─ 挑战更高难度
```

### 有基础用户
```
1️⃣ 游戏模式（培养节奏感）
   └─ 从简单开始，逐步提高难度
   
2️⃣ 主页-自动演奏（欣赏学习）
   └─ 听完整曲目，理解旋律
   
3️⃣ 主页-手动练习（深度学习）
   └─ 分段练习，掌握技巧
```

### 娱乐用户
```
🎮 直接玩游戏模式
   └─ 挑战高分，解锁成就
   └─ 和朋友PK排行榜
```

## ⚡ 快速提示

### 键盘操作通用映射
所有模式使用相同的键盘映射：
```
黑键: W E   T Y U   O P
白键: A S D F G H J K L ;
音符: C D E F G A B C D E
```

### 音频问题
如果听不到声音：
1. 检查设备音量是否开启
2. 确保浏览器没有静音
3. 在跟弹/游戏模式中，点击页面任意位置激活音频
4. 刷新页面重新加载

### 性能优化
- 关闭其他占用资源的标签页
- 使用现代浏览器（Chrome、Firefox、Edge）
- 游戏模式如果卡顿，可以尝试降低难度
