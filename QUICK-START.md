# 🚀 快速开始指南

## 🎓 零基础用户？从这里开始！

如果您是**完全零基础**，从未接触过钢琴，强烈推荐先使用**跟着弹模式**！

### 1️⃣ 打开跟着弹页面

**方式一：直接打开文件**
```
双击打开项目中的 follow-along.html 文件
```

**方式二：浏览器访问**
```
http://localhost:8080/follow-along.html
（需要先启动本地服务器）
```

### 2️⃣ 选择歌曲并开始

1. 从下拉菜单选择一首歌曲（推荐：千与千寻）
2. 点击"开始练习"按钮
3. 看到高亮的琴键，用鼠标点击或按键盘
4. 弹对了自动进入下一个音符
5. 完成所有音符即可！

### 3️⃣ 其他功能

**已经有基础？**
- 打开 `index.html` 使用完整功能
- 自动播放、MIDI、手动练习等

**想系统学习？**
- 查看 [使用指南](README_CN.md)
- 阅读 [跟着弹详细说明](FOLLOW-ALONG-GUIDE.md)

## 💡 小贴士

✅ **首次使用**
- 建议在WiFi环境下打开，需要加载音频
- 首次加载后会缓存，之后很流畅

✅ **最佳体验**
- 电脑：使用Chrome/Firefox/Edge浏览器
- 手机：横屏使用，视野更好

✅ **遇到问题**
- 刷新页面重试
- 检查浏览器是否允许播放音频
- 查看控制台是否有错误信息
