# 🚀 快速开始指南 | Quick Start Guide

欢迎使用在线钢琴练习平台！这是一个功能丰富的钢琴学习工具，提供多种学习模式。

## 📍 选择你的学习路径

### 🎮 游戏模式（推荐给喜欢玩游戏的你！）
**文件：** `game-mode.html`

像玩 Synthesia 或 Piano Tiles 一样学钢琴！
- 🎯 音符从上方落下，按键得分
- 🏆 三种难度选择
- 💯 追求 Perfect 判定和高连击
- 📊 成就系统和排行榜

**适合：** 喜欢游戏化学习、追求挑战的用户

[→ 点击进入游戏模式](game-mode.html)

---

### 🎓 跟着弹模式（推荐给零基础新手！）
**文件：** `follow-along.html`

一步一步学习，弹对才能继续！
- ✨ 琴键高亮提示
- ✅ 即时正误反馈
- 📈 进度条显示
- 🎵 简化的30音符练习

**适合：** 完全零基础、第一次学钢琴的用户

[→ 点击进入跟着弹模式](follow-along.html)

---

### 🎹 主程序（完整功能）
**文件：** `index.html`

包含所有功能的完整版本：
- 🎵 自由弹奏
- 🎼 自动演奏
- 📝 手动练习
- 🎹 MIDI 播放
- 🖼️ 随机壁纸
- 💝 更多功能...

**适合：** 想探索所有功能的用户

[→ 点击进入主程序](index.html)

## 🎯 推荐学习路径

### 路径 A：游戏化学习（有趣！）
```
游戏模式（简单难度）
    ↓
游戏模式（普通难度）
    ↓
跟着弹模式（细节练习）
    ↓
主程序自动演奏（完整曲目）
```

### 路径 B：稳扎稳打（推荐零基础）
```
跟着弹模式（学习基础）
    ↓
主程序手动练习（分步练习）
    ↓
游戏模式（测试技能）
    ↓
主程序自由弹奏（自由创作）
```

### 路径 C：快速上手（有基础）
```
主程序自动演奏（听完整曲目）
    ↓
游戏模式（快速练习）
    ↓
主程序自由弹奏（掌握技能）
```

## ⌨️ 键盘映射

所有模式都使用相同的键盘映射：

```
白键（主音符）：
键盘：A  S  D  F  G  H  J  K  L
音符：C4 D4 E4 F4 G4 A4 B4 C5 D5
唱名：Do Re Mi Fa Sol La Si Do Re

黑键（升降音）：
键盘：W  E     T  Y  U     O  P
音符：C# D#    F# G# A#    C# D#
```

## 💡 快速提示

### 第一次使用？
1. **手机用户：** 建议横屏使用，体验更好
2. **电脑用户：** 确保键盘功能正常
3. **所有用户：** 确保音量已打开

### 没有声音？
1. 检查设备音量
2. 刷新页面重试
3. 允许浏览器播放音频

### 想添加自己的歌曲？
查看项目的 README.md 了解如何添加自定义曲谱。
